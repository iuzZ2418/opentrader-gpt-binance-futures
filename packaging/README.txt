A股研究证据与观点跟踪工作台 v0.8.0

使用方法
1. 将整个 CompanyEventMonitor 文件夹解压到本地磁盘。
2. 双击 CompanyEventMonitor.exe。
3. 软件将直接打开Windows桌面窗口，不会启动浏览器或本地Web服务。
4. 在“查询公司”输入A股代码或简称；结果保存在“已查公司”。
5. 在“研究观点”建立需要持续验证的观点，查看支持、反对和分歧证据。
6. 在“公司比较”选择2—6家公司，软件会先更新公告与行情，再生成比较。
7. 在“设置”录入DeepSeek API密钥；密钥仅保存在Windows凭据管理器。

新增研究能力
- 首次查询默认回溯3年，材料不足扩展至5年，不再设置30份深处理硬上限。
- 交易所与巨潮多源公告检索、去重、正文解析和原文证据回溯。
- 研究观点自动关联支持/反对证据，可人工确认、失效或恢复自动判断。
- 管理层承诺兑现跟踪，以及同类文档新增/删除表述识别。
- 腾讯证券公开A股复权日线，计算多周期收益、相对指数、波动和回撤。
- 将标准化事件与公告后5/20交易日价格反应关联。
- 生成20交易日上行/震荡/下行情景、历史收益区间和滚动方向回测。
- 不输出买入、卖出、目标价或自动评级。

数据目录
%LOCALAPPDATA%\CompanyEventMonitor

说明
- 沪深公告优先使用交易所官网并以巨潮补充；北交所目前主要使用巨潮。
- 行情公开接口可能受网络或供应商限制；失败时仍可打开已有本地结果。
- 情景预测是研究辅助，不是收益承诺或投资建议。
